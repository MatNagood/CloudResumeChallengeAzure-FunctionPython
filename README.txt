Azure Function (v1) en Python pour lire et écrire sur une cosmosDB (noSQL api)
 - Les informations sur la cosmosDB sont dans le fichier function.json
 - La variable AzureCosmosDBConnectionString (correspondant à la clée de la cosmosDB) est à définir dans le fichier README.txt

Réalisé pour le Cloud Resume Challenge : https://cloudresumechallenge.dev/
